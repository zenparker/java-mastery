# Contributing to Java Mastery

Thank you for your interest in contributing to the Java Mastery project! This repository aims to provide a comprehensive learning path for Java programming, and your contributions are valuable in improving the educational content.

## How to Contribute

### Reporting Issues
- If you find errors, typos, or unclear explanations, please open an issue
- Describe the problem clearly and suggest improvements if possible
- Include the specific file and location where the issue occurs

### Adding Content
- Follow the existing folder structure and naming convention
- Ensure examples are clear, well-commented, and educational
- Maintain consistency with the existing teaching style
- Add practical examples that reinforce concepts

### Improving Existing Content
- Refactor confusing examples to make them clearer
- Add additional comments or explanations where helpful
- Improve code formatting and structure
- Add edge cases or advanced examples

## Code Style Guidelines

- Use meaningful variable and method names
- Write clear, descriptive comments
- Follow Java naming conventions (camelCase for variables/methods, PascalCase for classes)
- Maintain consistent indentation (use 4 spaces)
- Include examples that demonstrate the concept being taught

## Folder Structure

The repository follows a chronological learning path:
- Phase 1: Setup & Fundamentals
- Phase 2: Arrays & Math
- Phase 3: Methods & Collections
- Phase 4: OOP Quick
- Phase 5: Exception Handling & Extras
- Phase 6: Advanced Topics

Each phase contains days or weeks of content building upon previous knowledge.

## Pull Request Process

1. Fork the repository
2. Create a new branch for your contribution
3. Make your changes following the guidelines above
4. Test your changes if applicable
5. Submit a pull request with a clear description of what you've added/improved
6. Link any related issues in your pull request description

## Questions?

If you have questions about contributing, feel free to open an issue or contact the repository owner.

Thank you for helping make Java Mastery a better learning resource!
